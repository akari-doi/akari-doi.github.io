WordPress Q&A

サイトを作った際にアニメーションなどをつけれますか？

はい、つけられます！

WordPressサイトにアニメーションを追加する方法は主に2つあります。

方法1：JavaScriptを使ってアニメーションをつける

これまでに学習した wp_enqueue_script() を使って、JavaScriptファイルを読み込み、アニメーションを実装できます。

手順：

1. JavaScriptファイルを作成

テーマフォルダ内に js/animation.js というファイルを作成します。

// animation.js の例（vanilla JavaScript）
document.addEventListener('DOMContentLoaded', function() {
    // スクロールで要素をフェードイン
    const elements = document.querySelectorAll('.fade-in');
    
    const observer = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
            if (entry.isIntersecting) {
                entry.target.classList.add('active');
            }
        });
    });
    
    elements.forEach(element => {
        observer.observe(element);
    });
});
💡 jQueryを使う場合
WordPressでjQueryを使う場合は、特殊な書き方が必要です。通常の $ は使えないため、以下のように記述します。 
// jQueryを使う場合
jQuery(function($) {
    // この中では $ が使える
    $('.fade-in').on('click', function() {
        $(this).addClass('active');
    });
});

2. functions.phpでJavaScriptを読み込む

function my_enqueue_scripts() {
    // JavaScriptファイルを読み込む
    wp_enqueue_script(
        'my-animation',                              // ハンドル名
        get_stylesheet_directory_uri() . '/js/animation.js',  // ファイルのパス
        array(),                                     // 依存関係（jQueryに依存する場合はarray('jquery')）
        '1.0.0',                                     // バージョン
        true                                         // フッターで読み込む
    );
}
add_action('wp_enqueue_scripts', 'my_enqueue_scripts');

📖 公式ドキュメント：wp_enqueue_script()

3. CSSでアニメーションのスタイルを定義

/* style.cssに追加 */
.fade-in {
    opacity: 0;
    transform: translateY(30px);
    transition: opacity 0.6s ease, transform 0.6s ease;
}

.fade-in.active {
    opacity: 1;
    transform: translateY(0);
}

4. HTMLでクラスを指定

<div class="fade-in">
    <h2>スクロールするとフェードインします</h2>
</div>
💡 ポイント
このように、wp_enqueue_script()を使えば、独自のJavaScriptを安全に読み込んでアニメーションを実装できます。

方法2：プラグインを使う（もっと簡単！）

コードを書かなくても、プラグインを使えばブロックごとに簡単にアニメーションをつけられます。

Blocks Animation

  • ブロックエディタの各ブロックに簡単にアニメーションを追加
  • フェードイン、スライドイン、ズームなど様々なアニメーション効果
  • 管理画面から直感的に設定可能
